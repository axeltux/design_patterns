En este proyecto se muestran las diferentes implementaciones de los Patrones de diseño de software 
usando el lenguaje de programación Python v3.

Los patrones se encuentran divididos en 3 categorias:

- Comportamiento
- Creacionales
- Estructurales

Dentro de los patrones de diseño de Comportamiento se encuentran:

- Chain of Responsibility
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- Observer
- State
- Strategy
- Template
- Visitor

Dentro de los patrones Creacionales se encuentran:

- Singleton
- Prototype
- Builder
- Factory Method
- Abstract Factory

Dentro de los patrones Estructurales se encuentran:

- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy 

Para mas información consulte el libro:

"PYTHON PATRONES DE DISEÑO"
